# 介绍

> 本项目是纯个人兴趣，模仿月亮计划的边狱巴士做的。
> 
> 使用了pugiXML库读取xml配置文件

目前可以做到一个敌人一个玩家，开局敌人会根据技能权重随机技能，玩家会有两个根据权重随机的技能进行选择。

然后开始拼点，拼点胜利的一方攻击，如果失败的一方技能内有Unbreakable硬币，就会在行动队列内立刻插入一个单方面攻击的行动。

PS：速度目前只能看看，没实质用处。

目前特效：

> 烧伤：burn
> 
> 流血：bleed
> 
> 破裂：rupture
> 
> 沉沦：sink
> 
> 震颤：tremor
> 
> 呼吸法：breath

> 震颤引爆：tremor-explode（若敌人有混乱条，就消耗一层层数，使敌人最前一个混乱条数值前移 震颤强度 点。（私货：若没有混乱条，则消耗一层层数，直接造成 { 震颤强度 / 2 } 伤害））
> 
> 沉沦泛滥：sink-explode

> #### 私货
> 
> 烧伤爆发：tremor-explode（消耗所有烧伤层数，直接造成 { 层数 * 强度 } 点伤害）
